# WeirdJs （怪异的JS）

https://github.com/khno/weirdJs/issues

### 这是什么

利用边缘冷门知识点的讲解来带领你去了解你所不知道且有趣的代码
通过每（隔一）天一道前端题目的方式，分享给大家

### 意义

这些**反面教材**可以帮助您在日常开发中避开不正确的编码习惯和姿势，切记不要拿来干坏事
